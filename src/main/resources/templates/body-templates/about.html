Description

This application allows user to find charging points according to his preference.
There is possibility to find charging point which is closest to user's current location, all points in given radius, nearest point to specific address, all points in radius from specific address, all points in town and all in country.

Technology used to create and develop this application:

  • Java 8
  • Spring Boot
  • Spring MVC
  • Spring Web Services
  • Thymeleaf
  • Bootstrap
  • JavaScript
  • Google Maps API
  • H2
  • JUnit 5
